# 力扣数据规模对应算法

| Input Size        | Complexity   |
| :---------------- | :----------- |
| $50000+$          | $O(n)$       |
| $10^5\sim 5*10^5$ | $O(n\lg(n))$ |
| $1000\sim 5000$   | $O(n^2)$     |
| $200\sim500$      | $O(n^3)$     |
| $20\sim24$        | $O(2^n)$     |
| $12-$             | $O(n!)$      |
